# 🎓 港硕模拟器

**HONG KONG MASTER SIMULATOR**

一个数值养成网页游戏，模拟香港硕士留学的5个月生活。

## 🎮 玩法

- 5个月 × 4周 = 20周
- 每周有 **7个时间格**，自由安排活动
- 通过学习、工作、娱乐来提升数值
- 部分高级活动需要先解锁
- 随机事件可能改变你的命运
- 最终走向 10 种不同结局

## 📊 六大数值

| 🧠 智商 | 💖 情商 | 💼 工作技能 | 💰 金钱 | 😰 压力值 | 🗣️ 语言能力 |

## 🏁 结局线

- **打工人线**：南山必胜客 / 维港三语人 / 外包人生 / 鼠鼠我啊
- **网红线**：网红顶流 / 千粉主播 / 三天打鱼
- **特殊线**：百亿富翁 / 港式婚绿 / 压力爆炸

## 🚀 运行方式

直接用浏览器打开 `index.html` 即可游玩，无需安装任何依赖。
