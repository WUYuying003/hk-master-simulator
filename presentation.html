Value & Novelty
Why This Project Matters

🎭 Cultural Innovation

  • Digitized the "Watered-Down Master" subculture unique to HK postgrad students
  • 22 endings reflect real life paths — from "Nanshan Law" to "McDonald's Hashbrown Master"
  • No similar themed simulation game exists

📡 System Design

  • Signal Tree — meta-progression with 17 unlockable nodes
  • 4 starting life paths with unique stat bonuses
  • Dynamic success rates = base + Signal + stat bonus
  • GPA checkpoints with real academic pressure

⚙️ Technical

  • 4,466 lines, single HTML file, zero external dependencies
  • Full i18n framework (ZH/EN) with localStorage
  • RPG-style UI with CSS animation engine
  • PWA-ready, GitHub Pages deployed

Storyworld Integration
Building on PolyU Storyworld

🌐 venetanji/polyu-storyworld

A shared universe of 600+ character definitions contributed by SD5967 students. Each character has YAML-defined personality traits, development arcs, origin stories, and superpowers.

🔗 How We Used It

  • Character personality traits informed our 8-stat system design (IQ, EQ, Skill mapping)
  • Character development curves inspired the 3-phase progressive unlock mechanic
  • Origin stories from the shared universe became seeds for our 22 ending narratives
  • The collaborative world-building ethos shaped our multiplayer-friendly design philosophy

📡 Storyworld MCP

The storyworld-mcp server provides character contexts that can be used for AI-powered interactions.

  • Character impersonation API for dynamic NPC dialogues
  • Personality-driven event response generation
  • Potential for AI-powered weekly commentary
🔮 Future Potential
Deeper MCP integration could enable dynamic, AI-generated events based on storyworld character interactions — making each playthrough unique.
